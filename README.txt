2) diagram.bmp файл-диаграмма 3-х таблиц

3) файл Functions.sql
содержит в себе реализацию 2-х функций

get_segment_payment - функция считает приблизительную сумму платежей операции движения в диапозоне дат

get_full_payment - функция считает приблизительную сумму платежей операции движения в зависимости о параметров

в теле функции get_full_payment мы получаем диапазон дат и для получения суммы вызываем get_segment_payment 


4) файл tables.sql содержит в себе объявление 3-х таблиц для работы с этими функциями часть параметров закомментировано
